What we do

Any salesforce cusomizations work in point and click (or) code following industry wide best practices.

Point and click tools

Workflow, Process builder, Visual flow, Next Best Action(Strategy Builder).

Lightning Aura Component & Lightning Web Component

Develop great User Interfaces following best UX principles. Also provide test automation scripts using JEST.

Apex & Visualforce

Triggers, Schedule & Batch scripts, REST API Custom Endpoints & much more in Apex. Also develop and support any tasks in Visualforce.

CI & Devops tools

Setup and enhance CI tools for Salesforce using Jenkins, CircleCI, Gitlab, etc.

Debugging & Refactoring

Specialize in debugging any code and find the root cause for the reporting issue. Code refactoring done to run code in optimal limits by using best practices.

Integrations

Work undertaken to connect salesforce to external system using point-to-point (or) through middleware tools like Talend, Mulesoft, Informatica, Jitterbit.
